Remote Assistance

Optionally, we can work on your spreadsheet remotely.
Sometimes our strict security and non-disclosure agreement isn't enough. For example, if you're prohibited from sending your spreadsheet outside your company network. In that case, we can complete the necessary changes remotely. This is done over web conference call and you'll see every mouse click that happens, as it happens (and retain full control).
Please let us know up-front if you need your task completed remotely, so we can include the cost in the quote. This will increase the cost, so please only choose this option if absolutely necessary.
